# Temporizador en JavaScript

Este proyecto es una simple aplicación de temporizador que permite iniciar, pausar y restablecer el tiempo. Se ha construido utilizando HTML, CSS y JavaScript, y es un excelente ejemplo de cómo usar `setInterval`, `clearInterval`, y manipulación del DOM para crear una funcionalidad interactiva.

## Características

- **Iniciar**: Comienza a contar el tiempo desde cero en formato `HH:MM:SS`.
- **Pausar**: Detiene el temporizador en el tiempo actual.
- **Restablecer**: Reinicia el temporizador a `00:00:00`.

## Tecnologías

- **HTML**: Para la estructura del temporizador.
- **CSS**: Para el diseño y la disposición visual de los elementos.
- **JavaScript**: Para la funcionalidad del temporizador.

## Instalación

No es necesario instalar ningún software adicional. Solo necesitas un navegador web.
